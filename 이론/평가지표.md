# R-square score
회귀 모델의 성능 평가용 수치. 예측값이 실제값을 얼마나 잘 설명하는가
$$R^{2}=1-\frac{\sum_{i}y_{i}-\hat{y}_{i}}{\sum_{i}y_{i}-\bar{y}_{i}}
$$
$y_{i}$ : 실제 값
$\hat{y}_{i}$ : 예측 값
$\bar{y}_{i}$ : 실제 값 평균
$R^{2}=1$ : 모델의 예측 정확도 max


# Correlation Coefficient
두 변수 간의 선형적 관계의 강도와 방향을 나타내는 수치. 
**Pearson correlation coefficient :**
$$r=\frac{\sum(x_{i}-\bar{x})(y_{i}-\bar{y})}{\sqrt{\sum(x_{i}-\bar{x})^{2}\cdot \sum(y_{i}-\bar{y})^{2}}}
$$
$x_{i}$ : 입력값
$y_{i}$ : 실제값
$\bar{x}, \bar{y}$ : 각 변수의 평균

	r=1 : 양의 선형 관계,   r=0 : 관계성 없음,   r=-1 : 음의 선형 관계
